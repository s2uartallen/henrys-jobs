# 🎮 Henry's Jobs - Educational Block Building Game

An exciting, colorful web-based game designed for children to learn about different jobs and occupations while having fun building with blocks!

## 🎯 Game Overview

**Henry's Jobs** is an educational game that combines the fun of Minecraft-style block building with learning about 9 different career paths. Players complete objectives related to various jobs, earn money, and progress through levels while discovering interesting facts about different occupations.

## 🎮 How to Play

### Controls
- **A / Left Arrow** - Move left
- **D / Right Arrow** - Move right
- **W / Space / Up Arrow** - Jump
- **Mouse Click** - Place colorful blocks
- **R** - Clear all placed blocks

### Gameplay Loop
1. Select a job from the job menu on the left
2. Read the educational fact about that job
3. View the objectives you need to complete
4. Build structures with blocks to complete objectives
5. Earn money upon job completion!
6. Progress through all 9 jobs to master them all!
